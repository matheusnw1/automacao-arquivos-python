# Automação de Arquivos em Python

Script em Python que organiza automaticamente arquivos de uma pasta por tipo.

## Funcionalidades
- Organiza imagens, documentos, vídeos e músicas
- Cria pastas automaticamente
- Move arquivos para os diretórios corretos

## Tecnologias
- Python
- os
- shutil

## Como usar

1. Execute o script
2. Digite o caminho da pasta
3. Os arquivos serão organizados automaticamente
